# Food Delivery Performance Analysis

##  Project Overview
This project analyzes food delivery performance data to identify factors affecting delivery time, cost, and customer satisfaction.

##  Business Problem
Late deliveries negatively impact customer experience and ratings.  
This analysis explores:
- What influences delivery time?
- Does cost affect delivery speed?
- How does delivery performance impact ratings?

##  Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Exploratory Data Analysis (EDA)

##  Key Insights
- Higher preparation times significantly increase delivery time.
- Orders exceeding 45 minutes tend to receive lower ratings.
- Certain cuisine categories show greater variability in delivery duration.

##  Recommendations
- Monitor delivery times exceeding 40 minutes.
- Improve restaurant preparation workflow efficiency.
- Focus on high-variance cuisine types to reduce delays.
